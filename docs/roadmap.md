## Kooder 路线图

代码就是数据，Kooder 希望成为一个深入的从企业代码资产中挖掘出代码本身的数据价值，包括开发能效、安全管控、代码审计等。
同时我们还希望 Kooder 能成为企业代码深度语义分析的引擎，提升智能编码能力，提升开发效率。

Kooder 下一步工作：

### 代码即数据

* 对接多种代码源（Gitea、SVN 等）
* 对接企业多个代码源，实现统一的代码数据管理
* 实现代码搜索的权限管理，使用者只能搜索其有权限的仓库代码
* 提供代码敏感信息告警能力

### 提升工程效能

* 根据人员、时间段进行代码统计，量化开发人员贡献度

### 代码智能推荐系统

* 实现对代码的语义分析和索引
* 实现 Web 代码浏览时的引用跳转
* 代码查重
* 智能代码推荐
